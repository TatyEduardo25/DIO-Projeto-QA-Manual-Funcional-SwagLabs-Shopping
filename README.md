## 📌 Sobre o projeto

Este projeto foi desenvolvido com o objetivo de simular o dia a dia de um Analista de QA em ambiente ágil.

O sistema utilizado foi o **SwagLabs Shopping**, uma aplicação de e-commerce onde foram aplicadas práticas reais de:

- Levantamento de requisitos
- Escrita de User Stories
- Planejamento Scrum
- Criação de fluxo no Jira
- Planejamento de testes
- Testes manuais funcionais
- Registro de bugs
- Documentação de QA

---

## 🎯 Objetivo

Aplicar práticas de testes manuais funcionais em um projeto ágil utilizando ferramentas amplamente usadas no mercado.
